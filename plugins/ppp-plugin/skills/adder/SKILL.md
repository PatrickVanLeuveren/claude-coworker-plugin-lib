---
name: adder
description: Use this skill to add two numbers together.
---
To use this skill, simply provide two numbers as input, and the skill will return their sum. For example, if you input 5 and 3, the skill will return 8.
You can also use this skill to add more than two numbers by providing a list of numbers as input. For example, if you input [1, 2, 3, 4], the skill will return 10.
